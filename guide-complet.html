🪖 Raid Policier (Cop Raid)

Description : Vagues de policiers attaquent chaque membre (jusqu’à 24 vagues).

Objectif :

  • Tuer ≥ 50% des ennemis par vague
  • Maximiser points individuels et clan

Récompenses : Or, points, classement Top 10, récompense clan.

Pertes : Oui – troupes définitivement perdues

Risque : Faible à Modéré

Conseils Débutant :

  • Demander renfort avant vague 15+
  • Ne pas retirer troupes trop tôt

Conseils Expert :

  • Optimiser composition défense (tank devant, DPS derrière)
  • Synchroniser renforts massifs fin d’event

📈 Daily Crime Wave

Description : Cycle hebdomadaire avec sous-objectifs quotidiens.

Objectif :

  • Accumuler points via construction, entraînement, PvP

Récompenses : Or, ressources, bonus classement.

Pertes : Non (sauf PvP volontaire)

Risque : Aucun

Conseil Expert :

Ne pas utiliser accélérateurs hors journée correspondante.

🎰 Conquête du Casino / Maroni Lab

Description : Raid PvE contre bâtiment neutre puissant.

Objectif : Infliger max dégâts en raids successifs.

Récompenses : Bonus proportionnels aux dégâts infligés.

Pertes : Oui

Risque : Modéré à Élevé

Organisation :

  • R5 fixe horaire
  • R4 lancent raids en boucle
  • Membres toujours full marche

🏛️ Élection du Maire

Description : Capture et défense bâtiment central (PvP massif).

Objectif : Tenir le bâtiment pendant durée imposée.

Récompenses : Buffs, bonus stratégiques, pouvoir décisionnel.

Pertes : Oui

Risque : Très Élevé

Conseils :

  • Boucliers avant event
  • Rotation défense
  • Débutants en soutien arrière
